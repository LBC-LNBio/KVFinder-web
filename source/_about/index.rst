Issues
======

KVFinder-web project uses of GitHub for project management. Please help us by reporting any problems or requests.

For our KVFinder-web server, you can file on `Issue <https://github.com/LBC-LNBio/parKVFinder-webservice/issues>`_ a:

.. toctree::
    
    Bug Report <https://github.com/LBC-LNBio/parKVFinder-webservice/issues/new?assignees=&labels=&template=bug_report.md&title=>
    Request for a new feature <https://github.com/LBC-LNBio/parKVFinder-webservice/issues/new?assignees=&labels=&template=feature_request.md&title=>

For our PyMOL plugin client, you can file on `Issue <https://github.com/LBC-LNBio/parKVFinder-webservice/issues>`_ a:

On `Issues <https://github.com/LBC-LNBio/pyKVFinder/issues>`_ page, you can file a:

.. toctree::
    
    Bug Report <https://github.com/LBC-LNBio/PyMOL-KVFinder-web-Tools/issues/new?assignees=&labels=&template=bug_report.md&title=>
    Request for a new feature <https://github.com/LBC-LNBio/PyMOL-KVFinder-web-Tools/issues/new?assignees=&labels=&template=feature_request.md&title=>

Only make sure the bug or request has not already been reported. Click on “Search” and enter some keywords to search.

Thank you for helping us improve KVFinder-web!
